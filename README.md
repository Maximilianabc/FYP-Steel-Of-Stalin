## Features

- Multiplayer
- Fog of War
- Unique signaling mechanisms
- Basic settings

Note: This is the FYP version of the game, more features will be implemented in full version

- - - -

## Gamemodes
Items | Casual | Campaign
:--- | :---: | :---:
Map Generation | Random | Pre-defined
\# AI players can be adjusted | Yes | Yes
\# Allies can be adjusted | Yes | Pre-defined
Units and tech are carried over to next battle | No | Yes
